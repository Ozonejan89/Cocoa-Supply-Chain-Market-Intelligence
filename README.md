# West African Cocoa Supply & Global Market Intelligence Dashboard

An end-to-end data analytics project using Python web scraping, SQL data engineering, and Power BI to track West African cocoa supply shocks and global market volatility.

## 🛠️ The Technical Stack
* **Data Extraction:** Python (BeautifulSoup / Requests) for automated market price and weather tracking pipelines.
* **Storage & Engineering:** SQL Database Management (Data cleaning, handling references, and computing regional volume aggregations).
* **Data Modeling & Visualization:** Power BI & DAX (Advanced star-schema modeling, custom dynamic KPI cards, and trend analytics layouts).

## 💡 Key Analytical Insights
* **The Parabolic Price Spike:** The dashboard uncovers a stable baseline in global cocoa prices from 2020 through 2023, followed by a dramatic market jump starting in 2024, peaking over $12K/ton.
* **The Production Trigger:** By structuring regional production timelines side-by-side with prices, the analysis highlights severe volume drop-offs across primary West African nations (Côte d'Ivoire and Ghana).
* **The Environmental Link:** Granular tracking of weather data logs confirms severe climate volatility immediately preceding the harvest drops, proving a data-driven link between environmental shocks and global economic trends.
